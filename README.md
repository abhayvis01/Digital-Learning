# Lumina - Premium Digital Solutions

Welcome to the Lumina project repository. This is a premium digital agency website designed to showcase digital services, portfolio items, and provide a way for potential clients to get in touch.

## Project Overview

Lumina is built with a focus on modern aesthetics, user experience, and performance. It features a sleek, responsive design that works seamlessly across all devices.

### Key Features

*   **Modern & Premium Design:** Utilizes a sophisticated color palette, high-quality typography (Outfit font), and smooth animations.
*   **Responsive Layout:** Fully responsive interface ensuring a great experience on desktops, tablets, and mobile phones.
*   **Multi-page Structure:** Includes Home, About, Services, and Contact pages.
*   **Interactive Elements:** Features a mobile-responsive navigation menu, hover effects, and interactive cards.

## Technologies Used

*   **HTML5:** For semantic structure and content.
*   **CSS3:** For styling, including Flexbox and Grid layouts, custom variables, and animations.
*   **JavaScript:** For interactive functionality such as the mobile menu toggle.
*   **Font Awesome:** For scalable vector icons.
*   **Google Fonts:** Using the 'Outfit' font family for typography.

## Getting Started

To view the project locally:

1.  Clone or download this repository.
2.  Open `index.html` in your web browser.

## File Structure

*   `index.html`: The main landing page.
*   `about.html`: Information about the agency and team.
*   `services.html`: Details of services offered.
*   `contact.html`: Contact form and information.
*   `css/style.css`: Main stylesheet containing all styles and variables.
*   `js/script.js`: JavaScript file for interactivity.
*   `assets/`: Directory for images and other static assets.
